## 技术规范

### 通用规范

- **协议**: HTTPS
- **数据格式**: JSON
- **字符编码**: UTF-8
- **时间格式**: ISO 8601 (YYYY-MM-DDTHH:mm:ssZ)
- **分页**: 统一使用 page/pageSize 参数

### 认证方式

```http
Authorization: Bearer <access_token>
```

### 通用响应格式

```json
{
  "code": 200,
  "message": "操作成功",
  "data": {},
  "timestamp": "2024-01-19T10:30:00Z"
}
```

### 通用错误码

| 错误码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未授权 |
| 403 | 权限不足 |
| 404 | 资源不存在 |
| 429 | 请求频率超限 |
| 500 | 服务器内部错误 |

- Node.js 16+
- 数据库: PostgreSQL 13+
- 缓存: Redis 6+
- 文件存储: 阿里云OSS/AWS S3
- 地图服务: 高德地图API
